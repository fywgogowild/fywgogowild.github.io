# Claude Code 影片剪輯大補帖

## 你的硬體條件

| 項目 | 規格 | 影響 |
|------|------|------|
| 機器 | MacBook Pro 2015 Intel i5 | 跑得動，但較慢 |
| RAM | 8GB | 不能同時跑太多大任務 |
| Whisper 建議模型 | `base` 或 `medium` | large 會卡死 |
| 預期處理時間 | 1 分鐘影片約 5-15 分鐘 | 耐心等待 |

---

## 已安裝工具清單

| 工具 | 用途 |
|------|------|
| faster-whisper | 語音轉字幕 |
| ffmpeg | 影片轉檔、剪輯、特效、字幕燒入 |
| auto-editor | 自動去除靜音片段 |
| Pillow | 製作字卡圖片 |
| elevenlabs | AI 配音（需 API Key） |
| requests | 從 Pexels 下載 B-roll |

啟動環境指令：
```bash
source ~/video-env/bin/activate
cd ~/VideoProjects
claude
```

## 能做到的事 & 提示詞範例

### 1. 基本：語音轉字幕 + 9:16 輸出

```
幫我把 ~/Downloads/影片.mp4 轉成 9:16，
用 Whisper 自動產生中文字幕並燒入，
輸出到 ~/Downloads/
```

---

### 2. 開場字卡

```
幫我在 ~/Downloads/影片.mp4 開頭加一張 2 秒字卡：
- 黑底白字
- 主標題：「追憶華語樂壇」（大字，置中）
- 副標題：「那些年的聲音」（小字，主標題下方）
接上原影片，輸出成新檔案
```

---

### 3. 關鍵字橘色高亮字幕

```
幫我把 ~/Downloads/影片.mp4 的字幕做成逐句出現，
關鍵詞（例如：王菲、小胖）自動標橘色，
其他字白色，黑色外框，字幕在畫面下方
```

---

### 4. 自動去除靜音（緊湊剪輯）

```
幫我用 auto-editor 把 ~/Downloads/影片.mp4
靜音超過 0.5 秒的片段都去掉，
讓影片節奏更緊湊，輸出成新檔案
```

---

### 5. 電影感調色

```
幫我對 ~/Downloads/影片.mp4 做調色：
- 輕微暗角
- 對比 +15%
- 飽和度 +10%
- 色調偏暖（橘黃色系）
輸出成新檔案
```

---

### 6. 鏡頭推進效果（Ken Burns）

```
幫我在 ~/Downloads/影片.mp4 加上鏡頭緩慢推進效果，
從 1.0 倍慢慢推到 1.2 倍，整支影片都要有，
讓靜態畫面看起來有動感
```

---

### 7. 重點段落變速

```
幫我把 ~/Downloads/影片.mp4 做變速處理：
- 第 10 秒到第 15 秒放慢到 0.7 倍速（重點段落）
- 其他部分維持正常速度
輸出成新檔案
```

---

### 8. 背景音樂 + 人聲自動閃避

```
幫我把 ~/Downloads/背景音樂.mp3 加到 ~/Downloads/影片.mp4：
- 背景音樂音量設為 30%
- 有人說話時自動降到 10%（ducking 效果）
- 影片結束時音樂淡出
輸出成新檔案
```

---

### 9. 從 Pexels 下載 B-roll

```
幫我用 Pexels API 搜尋「台北夜景」相關影片，
下載 3 支最高畫質的，存到 ~/Downloads/broll/ 資料夾
```

---

### 10. ElevenLabs AI 配音

```
幫我用 ElevenLabs 把以下文字轉成語音，
使用繁體中文女聲，語速正常：
「這是一支關於記憶的影片，獻給所有愛音樂的人。」
存到 ~/Downloads/voiceover.mp3
```

---

### 11. 全自動一條龍（進階）

```
幫我對 ~/Downloads/影片.mp4 做完整處理：
1. 用 Whisper 轉字幕（繁體中文）
2. auto-editor 去除靜音
3. 加開場字卡「追憶華語樂壇」（黑底白字 2 秒）
4. 調色（暖色系、對比加強）
5. 字幕燒入（白字黑邊）
6. 輸出 9:16 成品到 ~/Downloads/final_output.mp4
```

## 注意事項

- **每次開新 session 都要先啟動虛擬環境**：`source ~/video-env/bin/activate`
- **Whisper 第一次用新模型會下載**，之後就快了
- **影片越長處理越久**，1 分鐘影片約 5-15 分鐘
- **同時只跑一個任務**，你的機器 RAM 不夠跑多個
- **輸出檔案預設在 ~/Downloads/**，除非你指定其他路徑
- **字型**：中文字幕預設用系統的 PingFang TC，不需要額外安裝

## 常見問題

**Q：字幕出現亂碼怎辦？**
```
字幕有亂碼，幫我重新辨識並指定繁體中文輸出
```

**Q：影片太大怎麼壓縮？**
```
幫我把 ~/Downloads/影片.mp4 壓縮到 50MB 以下，畫質盡量保留
```

**Q：想換字幕字型？**
```
幫我把字幕字型換成蘋方粗體，字級 24，黃色字白色外框
```

**Q：auto-editor 出錯？**
先在普通 Terminal 跑：
```bash
/Applications/Python\ 3.12/Install\ Certificates.command
```

## 工作流程建議

```
原始影片
   ↓
① 去除靜音（auto-editor）
   ↓
② 加開場字卡
   ↓
③ 調色
   ↓
④ 加字幕
   ↓
⑤ 加背景音樂
   ↓
⑥ 輸出 9:16
```

每個步驟都可以單獨做，不用一次全做。
