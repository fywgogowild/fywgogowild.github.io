Creating a New Theme

從 2018 年的北海道之旅後，馬上就去動物園辦了年卡，接下來的每一個六日，幾乎不間斷地到動物園拍照。原先純粹只是喜歡小貓熊，沒想到後來就認真喜歡上拍攝動物，應該說，是野生動物。

雖然台灣有得天獨厚的自然環境，山林間也有不同物種的生物，但作為動物攝影初心者，練習拍攝技術是現階段最重要的事，所以上網做了很多功課，第一步就是器材的準備。
