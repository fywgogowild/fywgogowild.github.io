其他平台（手動）
Facebook 貼文
到貼文頁面，右上角選「嵌入」，複製產生的 <iframe> 整段貼到 Source 模式。
TikTok
到影片頁面，點「分享 → 嵌入」，複製 <blockquote> + <script> 整段貼到 Source 模式。
Vimeo
到影片頁面，點「分享 → 嵌入」，複製 <iframe> 整段貼到 Source 模式。
Spotify
到歌曲／播放清單，點「⋯ → 分享 → 嵌入」，複製 <iframe> 整段貼到 Source 模式。
Google Maps
到地點，點「分享 → 嵌入地圖」，複製 <iframe> 整段貼到 Source 模式。
